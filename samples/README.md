# Pasta de Imagens de Teste

Coloque aqui suas imagens para teste:

- `selfie.jpg` - Foto de selfie
- `cnh.jpg` - Foto da CNH

## Requisitos das Imagens

- **Formato**: JPG, JPEG ou PNG
- **Tamanho máximo**: 5MB
- **Resolução mínima**: 640x480
- **Face visível**: Frontal, bem iluminada
- **Qualidade**: Boa resolução, sem blur

## Onde Obter Imagens de Teste

1. Tire uma selfie com boa iluminação
2. Fotografe sua CNH (frente)
3. Ou use imagens de teste públicas (com permissão)
